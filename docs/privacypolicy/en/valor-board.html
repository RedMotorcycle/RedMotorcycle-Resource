Valor Board Privacy Policy

Last Updated: January 14, 2026

By using this application, you agree to this Privacy Policy.

Users are asked to confirm their agreement to the Terms of Use when installing or updating the application.

Age Restriction

This application is intended for users aged 16 and older. This complies with Riot Games' account creation requirements. Users under 16 must obtain parental consent before use.

Data We Collect and Use

Data Obtained via Riot Games API

  • Riot ID (Game Name #Tag Line)
  • Account Information (Identifiers like PUUID)
  • Match History and Performance Data
  • Rank Information

This application retrieves and displays player match history and rank information via the Riot Games API.

To view personal match data (match history, detailed stats, dashboard features, etc.), we require user authentication (Opt-in) via Riot Sign On (RSO) to verify data ownership and protect privacy.

By authenticating through RSO, you agree (Opt-in) that your player data will be retrieved, analyzed, and displayed by this application.

Data Saved on Your Device

  • Authentication Tokens (Saved in secure encrypted storage)
  • User Settings and Application Configuration
  • Cache Data (Game content information, etc.)
  • Images and videos used for the Lineup feature (stored only on the device and not sent to external servers)

Data Sent to Servers

  • Firebase Analytics: App usage data (contains no personally identifiable information)
  • Firebase Crashlytics: Crash reports (includes device information)
  • Cloud Firestore: User PUUID and consent information (for data validation among app users)
  • Firebase App Check: Tokens for verifying app and device authenticity (Google Play Integrity API / Apple DeviceCheck / App Attest)
  • Google AdMob: Advertising identifiers (IDFA/AAID), device information, and interaction data (for ad delivery and analysis)

Data Retention and Deletion

Data Storage Locations

  • Device Storage: Authentication info, settings, cache
  • Firebase Servers (Google Cloud): Analytics data, crash reports
  • Cloud Firestore (Google Cloud / Tokyo Region): Opt-in user identifiers (PUUID)
  • Google AdMob Servers: Advertising-related data

This application does not collect data on proprietary servers but uses the aforementioned Firebase services and Google AdMob to provide features.

Data Retention Period

  • Device Data: Until the app is uninstalled
  • Firebase Analytics: Subject to Google's data retention policies
  • Firebase Crashlytics: 90 days
  • Google AdMob: Subject to Google's data retention policies

How to Delete Data

Data on the device is deleted by uninstalling the app. Data sent to Firebase and AdMob is processed according to Google's policies.

User Rights and Opt-out

Users have the following rights regarding their data:

  • Revoking consent for data collection (Opt-out)
  • Complete deletion of device data
  • Opting out of personalized ads (via device settings)

Exercising Rights and Deleting Data

You can stop data usage and delete data by performing the following actions:

  • Logging out of the App (Disconnecting RSO):
    Please log out from the settings screen within the app. By selecting the 'Hide stats' option when logging out, you can delete your opt-in registration on the server. If you do not select this option, your data will remain searchable.
  • Uninstalling the App:
    All data on your device (authentication tokens, settings, cache) will be completely deleted. Transmission of statistical data like Firebase Analytics will also stop.

Security Measures

  • Authentication tokens are stored in secure encrypted storage on the device (iOS: Keychain, Android: EncryptedSharedPreferences).
  • Communication with external servers is encrypted using HTTPS.
  • Confidential information such as authentication tokens is not shared with third parties.
