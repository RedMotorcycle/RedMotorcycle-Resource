Valor Board プライバシーポリシー

最終更新日: 2026年1月2日

当アプリのご利用に当たって、以下のプライバシーポリシーにご理解頂いた上で利用することに同意するものとします。

ユーザーの皆様にはアプリのインストールおよびアップデート時に利用規約に同意するための画面をご確認いただいています。

対象年齢

当アプリは16歳以上の方を対象としています。これはRiot Gamesのアカウント作成要件に準拠しています。16歳未満の方は保護者の同意を得た上でご利用ください。

収集・利用するデータ

Riot Games APIを通じて取得するデータ

  • Riot ID（ゲームネーム#タグライン）
  • アカウント情報（PUUID等の識別子）
  • マッチ履歴・戦績データ
  • ランキング情報

当アプリでは、Riot Games APIを通じて一般に公開されている個別のマッチ履歴やランク情報などを取得・表示します。

ただし、これらのデータを集計・分析した詳細な統計情報（ダッシュボード機能等）の閲覧には、データの主体性を確認し保護するため、Riot Sign On (RSO) によるご本人様の認証（オプトイン）を必須としています。

端末に保存するデータ

  • 認証トークン（安全な暗号化ストレージに保存）
  • ユーザー設定・アプリ設定
  • キャッシュデータ（ゲームコンテンツ情報等）

サーバーに送信するデータ

  • Firebase Analytics: アプリ使用状況（個人特定情報は含まれません）
  • Firebase Crashlytics: クラッシュレポート（デバイス情報含む）

データの保存と削除

データの保存場所

  • 端末内ストレージ: 認証情報、設定、キャッシュ
  • Firebaseサーバー（Google Cloud）: 分析データ、クラッシュレポート

当アプリは独自のサーバーにユーザーデータを保存・収集することはありません。

データの保持期間

  • 端末内データ: アプリをアンインストールするまで
  • Firebase Analytics: Googleのデータ保持ポリシーに従う
  • Firebase Crashlytics: 90日間

データの削除方法

端末内のデータはアプリをアンインストールすることで削除されます。Firebaseに送信されたデータについてはGoogleのポリシーに従って処理されます。

ユーザーの権利

ユーザーは以下の権利を有します：

  • データ収集へのオプトアウト（アプリのアンインストール）
  • 端末内データの完全削除（アプリのアンインストール）

セキュリティ対策

  • 認証トークンは端末の安全な暗号化ストレージ（iOS: Keychain、Android: EncryptedSharedPreferences）に保存されます
  • 外部サーバーへの通信はHTTPSを使用し暗号化されています
  • Riot ID以外の認証情報は第三者と共有されません

第三者サービス

当アプリは以下の第三者サービスを利用しています：

Riot Games API

VALORANTのゲームデータを取得するために使用します。

  • Riot Games プライバシーノーティス
  • Riot Games 利用規約

Firebase (Google)

アプリの分析とクラッシュレポートに使用します。

  • Google プライバシーポリシー
  • Firebase プライバシー情報

プライバシーポリシーの変更

本ポリシーは必要に応じて更新されることがあります。アプリのアップデート後、初回起動時に新しいポリシーへの同意をお願いする場合があります。
