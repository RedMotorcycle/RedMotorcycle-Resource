Valor Board 隱私權政策

最後更新日期：2026年1月9日

在使用本應用程式之前，請務必閱讀並同意本隱私權政策。

用戶在安裝或更新應用程式時，會被要求確認並同意使用條款。

年齡限制

本應用程式適用於16歲及以上的用戶。這符合 Riot Games 帳戶建立的要求。未滿16歲的用戶請在獲得監護人同意後使用。

收集與使用的資料

透過 Riot Games API 取得的資料

  • Riot ID（遊戲名稱 #標籤）
  • 帳戶資訊（PUUID 等識別碼）
  • 對戰紀錄與戰績資料
  • 排名資訊

本應用程式透過 Riot Games API 取得並顯示玩家的對戰紀錄與排名資訊。

為了查看個人的戰績資料（對戰紀錄、詳細統計、儀表板功能等），我們要求透過 Riot Sign On (RSO) 進行用戶驗證（Opt-in），以確認資料所有權並保護隱私。

透過 RSO 進行驗證，即視為您同意（Opt-in）本應用程式取得、分析並顯示您的玩家資料。

儲存在裝置上的資料

  • 驗證權杖（儲存在安全的加密儲存空間）
  • 用戶設定與應用程式設定
  • 快取資料（遊戲內容資訊等）

傳送至伺服器的資料

  • Firebase Analytics：應用程式使用狀況（不含個人識別資訊）
  • Firebase Crashlytics：崩潰報告（包含裝置資訊）
  • Cloud Firestore：用戶的 PUUID 及同意資訊（用於應用程式用戶間的資料驗證）
  • Firebase App Check：用於驗證應用程式與裝置真實性的權杖（Google Play Integrity API / Apple DeviceCheck / App Attest）
  • Google AdMob：廣告識別碼（IDFA/AAID）、裝置資訊及互動資料（用於廣告投放與分析）

資料的保存與刪除

資料儲存位置

  • 裝置內部儲存：驗證資訊、設定、快取
  • Firebase 伺服器（Google Cloud）：分析資料、崩潰報告
  • Cloud Firestore（Google Cloud / 東京區域）：Opt-in 用戶的識別碼（PUUID）
  • Google AdMob 伺服器：廣告相關資料

本應用程式不會在自有伺服器上收集資料，而是利用上述 Firebase 服務及 Google AdMob 來提供功能。

資料保存期間

  • 裝置內部資料：直到解除安裝應用程式為止
  • Firebase Analytics：遵循 Google 的資料保存政策
  • Firebase Crashlytics：90天
  • Google AdMob：遵循 Google 的資料保存政策

資料刪除方法

裝置內的資料可透過解除安裝應用程式來刪除。傳送至 Firebase 及 AdMob 的資料將根據 Google 的政策進行處理。

用戶權利與拒絕（Opt-out）

用戶對於自身資料擁有以下權利：

  • 撤回資料取得的同意（解除 Opt-in）
  • 完全刪除裝置內資料
  • 拒絕個人化廣告（透過裝置設定）

行使權利及刪除資料的方法

您可以透過以下操作停止資料使用並刪除資料：

  • 從應用程式登出（解除 RSO 連結）：
    請從應用程式內的設定畫面進行登出。在登出時選擇「隱藏數據」選項，您可以刪除伺服器上的加入（Opt-in）註冊資訊。如果您不選擇此選項，您的數據將保持可搜索狀態。
  • 解除安裝應用程式：
    裝置內的所有資料（驗證權杖、設定、快取）將被完全刪除。Firebase Analytics 等統計資料的傳送也會停止。

資安措施

  • 驗證權杖儲存在裝置的安全加密儲存空間中（iOS: Keychain, Android: EncryptedSharedPreferences）。
  • 與外部伺服器的通訊均使用 HTTPS 進行加密。
  • 驗證權杖等機密資訊不會與第三方共享。

第三方服務

本應用程式使用以下第三方服務：

Riot Games API

用於取得 VALORANT 遊戲資料。

  • Riot Games 隱私權聲明
  • Riot Games 服務條款

Firebase (Google)

用於應用程式分析、崩潰報告及安全性保護（防止不正當使用）。

  • Google 隱私權政策
  • Firebase 隱私權資訊

Google AdMob

用於廣告投放。可能會收集廣告識別碼及裝置資訊，用於向用戶顯示合適的廣告。

  • Google 廣告 – 政策與條款
